1. 什么是 MCP？

MCP（Model Context Protocol） 是 AI 世界的 "USB 接口"——让 AI Agent 能直接调用外部工具和数据源。

没有 MCP：你问 Claude "川润股份最近有减持公告吗？"
Claude 回答："我无法获取实时数据，建议你去巨潮资讯网查询。"

有了 AStockEvent MCP：同一个问题
Claude 自动调用 search_events_by_stock → 返回结构化结果 → "川润股份(002272) 6月8日发布减持预披露公告，股东XXX计划减持不超过X股，当前处于plan阶段，可信度 verified。"

2. 安装（30 秒）

需要 Python 3.11+。一行命令：

pip install astockevent

验证安装：

python -c "import astockevent; print(astockevent.__version__)"
# → 0.2.1

3. 额度说明

无需注册，开箱即用。 当前每天可调用 100 次，个人使用完全够用。

如果需要更高额度，可以添加 ASTOCKEVENT_API_KEY 环境变量（注册功能即将上线，注册后额度 200 次/天）。下方的配置示例已经是即复制即用的最简版本——不需要填任何 Key。

4. 接入 Claude Desktop

打开 claude_desktop_config.json（位置见下），添加：

{
  "mcpServers": {
    "astockevent": {
      "command": "python",
      "args": ["-m", "astockevent.mcp"]
    }
  }
}
系统配置文件路径
macOS~/Library/Application Support/Claude/claude_desktop_config.json
Windows%APPDATA%\Claude\claude_desktop_config.json

重启 Claude Desktop 后生效。工具栏会出现 🔌 图标，点击可查看 16 个工具。

5. 接入 Hermes Agent

编辑 ~/.hermes/config.yaml，在 mcp_servers: 下添加：

mcp_servers:
  astockevent:
    command: "python"
    args: ["-m", "astockevent.mcp"]

重启 Hermes 或执行 /reload-mcp 即可生效。

6. 接入 Cursor

Cursor 设置 → MCP → 添加新 Server：

{
  "mcpServers": {
    "astockevent": {
      "command": "python",
      "args": ["-m", "astockevent.mcp"]
    }
  }
}

7. 接入 VSCode Copilot

在 VSCode 设置中搜索 mcp，或直接编辑 .vscode/mcp.json：

      "astockevent": {

10. 常见问题

Q: 网络需要特殊配置吗？
不需要。MCP Server 在本地运行，通过 HTTPS 通信，国内网络直连可用。
Q: 数据从哪里来？
来自巨潮资讯网、深交所、东方财富等 A 股官方信息披露渠道，多源交叉验证确保准确性。
Q: 支持哪些 AI Agent？
Claude Desktop、Hermes Agent、Cursor、VSCode Copilot，以及任何支持 MCP stdio 协议的 AI 客户端。
Q: REST API 和 MCP 有什么区别？
REST API 适合程序代码直接调用（Python/JS/Go HTTP 请求）。MCP 适合 AI Agent 场景——Agent 自动发现工具、自动调用，你只需要用自然语言提问。两者使用同一数据源。
Q: 需要注册吗？
不需要。当前开箱即用，每日 100 次调用。注册功能上线后，注册用户可获更高额度。
